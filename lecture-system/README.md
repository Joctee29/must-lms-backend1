# MUST LMS - Lecture System

## Instructor/Lecturer Portal

This is the instructor portal for the MUST Learning Management System. It provides tools for course creation, content management, student assessment, and teaching analytics.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The system will run on http://localhost:3000

## Features

- Course Creation & Management
- Content Upload & Organization
- Live Session Management
- Student Assessment Tools
- Grade Management
- Analytics & Reports
- Communication Tools

## System Role: Instructor/Lecturer

This system is specifically designed for educators to:
- Create and manage courses
- Upload learning materials
- Conduct live sessions
- Assess student performance
- Track learning progress
- Communicate with students
