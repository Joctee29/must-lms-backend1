# MUST LMS - Admin System

## Administrative Management Portal

This is the administrative portal for the MUST Learning Management System. It provides comprehensive system management, user administration, and analytics oversight.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The system will run on http://localhost:3002

## Features

- User Management (Students, Instructors, Admins)
- Course Approval & Management
- System Configuration
- Analytics & Reporting
- Bulk Operations
- Content Moderation
- System Monitoring
- Certificate Management

## System Role: Administrator

This system is specifically designed for administrators to:
- Manage users and permissions
- Oversee course catalog
- Monitor system performance
- Generate comprehensive reports
- Configure system settings
- Manage institutional policies
