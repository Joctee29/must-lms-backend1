# MUST LMS - Student System

## Student Learning Portal

This is the student portal for the MUST Learning Management System. It provides access to courses, learning materials, assessments, and progress tracking.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The system will run on http://localhost:3001

## Features

- Course Enrollment & Access
- Learning Materials & Resources
- Assignment Submission
- Quiz & Assessment Taking
- Progress Tracking
- Grade Viewing
- Discussion Forums
- Certificate Generation

## System Role: Student/Learner

This system is specifically designed for students to:
- Browse and enroll in courses
- Access learning materials
- Complete assignments and assessments
- Track learning progress
- Communicate with instructors
- View grades and certificates
